# Aplicacion usando La api Usda Food data

Esta aplicacion permite buscar alimentos y obtener informacion nutricional usando la API de USDA Food Data Central.

## Caracteristicas
- Buscar alimentos por nombre o categoria.
- Ver detalles nutricionales de los alimentos.
- Guardar alimentos favoritos para referencia futura.
